# voiceChatWithQianwen
This project is to implement a voice version chatbot with large language model Qianwen in Ubuntu systemF
Features: User(human) speak through microphone(either PC or embedded chip). Large language Model(LLM) will 
output(speak） through speaker(either PC or embedded chip). When LLM speak too long, User(human) can interrupt the
LLM in speaker and start a new round of chat.
